# Pollyana Valença — Advocacia

Landing page de Pollyana Valença, advogada (OAB/TO 13.057) atuando em Direito Civil, Previdenciário, Trabalhista e de Família em Tocantins. Inclui uma seção separada para escritórios de advocacia que precisam de correspondente jurídica (protocolos, cópias e diligências) em Guaraí, Palmas e Porto Nacional.

## Stack

HTML, CSS e JS puros — sem build, sem dependências. Um único arquivo `index.html`.

## Deploy

Projeto pronto para deploy estático no [Vercel](https://vercel.com):

1. Suba este repositório
2. No Vercel, "Add New" → "Project" → importe o repositório
3. Deploy (não precisa configurar nada)

## Editar

- **Foto/logo:** procure os comentários `<!-- FOTO: ... -->` e `<!-- LOGO: ... -->` no `index.html` para saber onde trocar os placeholders por imagens reais.
- **WhatsApp/contato:** o número está fixo nos links `https://api.whatsapp.com/send?phone=...`; troque em todas as ocorrências se mudar o número.
